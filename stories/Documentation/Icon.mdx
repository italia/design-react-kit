# Icone

### Informazioni e suggerimenti per l’utilizzo di icone con Bootstrap Italia.

Le icone di Bootstrap Italia utilizzano sono state incapsulate in un componente `Icon`, e vengono caricate unicamente (in modalià asincrona) su richiesta quando si utilizza la nuova versione ESM.

## Esempi

Ogni icona può essere inclusa con il componente `Icon` con il riferimento desiderato:

## Dimensioni

È possibile utilizzare il modificatore `size` con il valori `xs`, `sm`, `lg`, `xl`, per ottenere icone di diverse dimensioni.

<Canvas of={IconStories.EsempioDimensioni} />

È anche possibile applicare il modificatore `padding` all’icona per creare un padding proporzionale alla dimensione dell’icona attorno ad essa.
