## Chiusura

È possibile eliminare qualsiasi avviso mediante un pulsante di chiusura:

-   Definisci una funzione che chiuda l'alert controllando la sua prop `isOpen`
-   Passa la funzione alla prop `toggle` del componente

Clicca sul pulsante di chiusura per vedere la funzionalità di rimozione altert in azione:

<Canvas of={AlertStories.ChiusuraControllata} />

#### Codice

```tsx
const [open, setOpen] = useState(true);

const closeAlert = () => setOpen(false);

return (
    <div>
        <Alert {...args} isOpen={open} toggle={closeAlert}>
            <strong>Attenzione</strong>Alcuni campi inseriti sono da controllare.
        </Alert>
    </div>
);
```
