# Sidebar

## Barra di navigazione laterale, include liste di link e liste di link innestate.

### Basic sidebar

#### Default

La sidebar semplice può contenere:

- Header
- Lista di link primaria
- Lista di link secondaria

Per differenziare a livello stilistico i link secondari, è sufficiente aggiungere l'attributo `secondary` al componente `Sidebar` contenente la [lista di link](?path=/docs/documentazione-organizzare-i-contenuti-linklist).

<Canvas of={SidebarStories.BasicSidebar} />

#### Con icona

La sidebar può contenere icone nella sua [lista di link](?path=/docs/documentazione-organizzare-i-contenuti-linklist) primaria.

<Canvas of={SidebarStories._ConIcona}>
  <Story id='componenti-sidebar--con-icona' />
</Canvas>

#### Con linea a destra

Per creare una sidebar con linea separatrice a destra è sufficiente aggiungere l'attributo `right` al componente `Sidebar`.

<Canvas of={SidebarStories.ConLineaADestra} />

#### Con linea a sinistra

Per creare una sidebar con linea separatrice a sinistra è sufficiente aggiungere l'attributo `left` al componente `Sidebar`.

<Canvas of={SidebarStories.ConLineaASinistra} />

### Annidata

#### Default

La sidebar può contenere una [lista di link](?path=/docs/documentazione-organizzare-i-contenuti-linklist) primaria annidata.

<Canvas of={SidebarStories._Annidata} />

### Sidebar versione scura

Per cambiare il tema della sidebar e renderla scura è sufficiente aggiungere al componente `Sidebar` l'attributo `dark`.
