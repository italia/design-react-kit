# Carousel

Il componente Carousel permette di scorrere una serie di contenuti ed è
realizzato attraverso il componente [Splide](https://splidejs.com/integration/react-splide/).

Può essere utilizzato con una serie di immagini, testo o codice personalizzato.
Include anche il supporto per i controlli e gli indicatori precedente/successivo
ed i dots di indicazione posizione.

Il carousel può contenere vari tipi di componenti, solitamente cards o immagini.

L'implementazione corrente riprende le varie tipologie di Carousel che possono
essere trovate in [Bootstrap Italia](https://italia.github.io/bootstrap-italia/docs/componenti/carousel/).

È possibile utilizzare le stesse proprietà del componente Splide nel Carousel
di questo kit.
