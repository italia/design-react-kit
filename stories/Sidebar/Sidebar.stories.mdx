# Sidebar

## Barra di navigazione laterale, include liste di link e liste di link innestate.

### Basic sidebar

#### Default

La sidebar semplice può contenere:

- Header
- Lista di link primaria
- Lista di link secondaria

Per differenziare a livello stilistico i link secondari, è sufficiente aggiungere l'attributo `secondary` al componente `Sidebar` contenente la <LinkTo kind="documentazione-organizzare-i-contenuti-linklist--page">lista di link</LinkTo>.

<Canvas>
  <Story id='componenti-sidebar--basic-sidebar' />
</Canvas>

#### Con icona

La sidebar può contenere icone nella sua <LinkTo kind="documentazione-organizzare-i-contenuti-linklist--page">lista di link</LinkTo> primaria.

<Canvas>
  <Story id='componenti-sidebar--con-icona' />
</Canvas>

#### Con linea a destra

Per creare una sidebar con linea separatrice a destra è sufficiente aggiungere l'attributo `right` al componente `Sidebar`.

<Canvas>
  <Story id='componenti-sidebar--con-linea-a-destra' />
</Canvas>

#### Con linea a sinistra

Per creare una sidebar con linea separatrice a sinistra è sufficiente aggiungere l'attributo `left` al componente `Sidebar`.

<Canvas>
  <Story id='componenti-sidebar--con-linea-a-sinistra' />
</Canvas>

### Annidata

#### Default

La sidebar può contenere una <LinkTo kind="documentazione-organizzare-i-contenuti-liste-di-link--page">lista di link</LinkTo> primaria annidata.

<Canvas>
  <Story id='componenti-sidebar--annidata' />
</Canvas>

### Sidebar versione scura

Per cambiare il tema della sidebar e renderla scura è sufficiente aggiungere al componente `Sidebar` l'attributo `dark`.
